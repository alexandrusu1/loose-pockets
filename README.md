# Loose Pockets

**Loose Pockets** este un mod pentru The Binding of Isaac: Repentance care face ca item-urile pasive să sară din inventar când primești damage! Ai la dispoziție câteva secunde să le recuperezi, altfel dispar pentru totdeauna. 

## Instalare
1. Copiază folderul `loose-pockets` în `mods` din directorul jocului.
2. Pornește jocul și activează mod-ul din meniul de MODS.

## Concept
- Când primești damage, pierzi 1-2 item-uri pasive pe jos.
- Item-urile pot fi recuperate rapid, altfel dispar.
- Lista neagră: item-uri de poveste nu pot fi pierdute.

## Dezvoltare
- main.lua: logica modului
- metadata.xml: metadate mod
- resources/: asseturi (sprites, sunete)
