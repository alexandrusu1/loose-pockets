Thank you for your interest in contributing to Loose Pockets.

How to contribute

- Open an issue to discuss larger changes before implementing them.
- Fork the repository and create a feature branch for your changes.
- Keep PRs small and focused; include a clear description and testing steps.

Coding style

- Use clear, idiomatic Lua. Follow the style used in `main.lua`.
- Keep functions short and single-responsibility.

Reporting bugs

- Provide game version, steps to reproduce, and expected vs actual behavior.

License

By contributing, you agree that your contributions will be licensed under the project's MIT license.
